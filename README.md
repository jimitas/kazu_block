# ぶろっく

小学1年生向けの**かずのがくしゅう**アプリです。ぶろっくをドラッグ&ドロップで並べながら、数の概念を視覚的に学べます。

![対象学年](https://img.shields.io/badge/対象-小学1年生-green) ![ライセンス](https://img.shields.io/badge/license-MIT-orange)

## 🎯 対象

- **小学1年生（4月入学当初）**を主な対象とした算数学習アプリ
- 数の概念をブロックを並べる・数えることで視覚的に学ぶ
- タブレット（iPad等）での使用を推奨

## ✨ 3つのモード

### ① なんこならべたかな？
ぶろっくを自由に並べて、「たしかめ」ボタンを押すと今いくつ並んでいるか教えてくれます。

### ② ならべたかずはいくつ？
「もんだい」を押すと、上段にぶろっくがランダムに並びます。
数字ボタン（0〜20）で答えを選ぶと正誤判定します。

### ③ ならべよう
「もんだい」を押すと「○こならべましょう」と問題が出ます。
ぶろっくのはこから上段のエリアにぶろっくをドラッグして並べ、「たしかめ」で正誤判定します。

### 🎚️ 難易度（モード②・③）
- **１〜５**
- **１〜１０**
- **１１〜２０**
- **１〜２０**

## 🧩 ぶろっくエリアの構成

```
┌──────────────┐  ┌──────────────┐
│  ならべるところ │  │  ならべるところ │  ← 上段（カウント対象）
└──────────────┘  └──────────────┘
━━━━━━━━━ ぶろっくのはこ ━━━━━━━━━
┌────────────────────────────────┐
│  ┌──────────┐  ┌──────────┐   │  ← 下段（木箱）
│  │ ピンク×10 │  │  青×10   │   │
│  └──────────┘  └──────────┘   │
└────────────────────────────────┘
```

- **上段**：ならべるエリア（ここのぶろっく数をカウント）
- **下段**：ぶろっくのはこ（ピンク10個・青10個でスタート）
- **リセット**ボタンで初期配置に戻る

## 🎵 効果音

- ぶろっくを動かしたとき
- モード・難易度を切り替えたとき
- 正解・不正解のフィードバック

## 📦 セットアップ

```bash

git clone https://github.com/jimitas/kazu_block.git
cd kazu_block

# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev
```

ブラウザで `http://localhost:3000` を開く。

```bash
npm run build
npm start
```
