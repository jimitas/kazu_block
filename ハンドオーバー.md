# block_1 プロジェクト ハンドオーバー

## プロジェクト概要

- **アプリ名（仮）**: ぶろっく
- **対象**: 小学1年生・4月入学当初
- **目的**: 数の概念をブロックを並べる・数えることで視覚的に学ぶアプリ
- **ベース**: `tashizan_1`（足し算練習アプリ）からコピーして作成

## tashizan_1 との違い

| 項目 | tashizan_1 | block_1 |
|---|---|---|
| 数式表示 | あり | **なし** |
| 難易度選択 | あり | **なし** |
| 答え合わせ（入力） | あり | **なし**（別の形で正誤判定） |
| ブロック機能 | 補助 | **メイン** |

## 実装予定の3機能

### ①確かめる機能
- 児童が並べたブロックの個数を数えて表示する
- 「いま○こ並んでいるよ」のようなフィードバック

### ②問題を出す機能
- 「４こ　ならべましょう」のように問題を出す
- 児童がブロックを並べた結果の個数が合っていれば正解とする
- 正誤の効果音あり

### ③自動で並べる機能
- 「４こ」と指定すると、自動でブロックを４個配置してくれる
- 手本を見せる・確認のために使う
